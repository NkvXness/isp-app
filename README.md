# Интернет-провайдер — Лабораторная работа №4

ASP.NET Core MVC приложение для учёта абонентов интернет-провайдера. Вариант 11. Entity Framework Core, MS SQL Server.

## Структура базы данных

| Таблица         | Описание                                    |
| --------------- | ------------------------------------------- |
| Tariffs         | Тарифные планы (домашние и офисные)         |
| Subscribers     | Абоненты провайдера                         |
| Equipment       | Оборудование абонентов (роутеры, приставки) |
| Payments        | Платежи абонентов                           |
| SupportRequests | Заявки в техподдержку                       |

## Функциональность

- Просмотр списков по всем таблицам
- Список абонентов с отрицательным балансом
- Статистика популярности тарифов
- Список незакрытых заявок в техподдержку
- Кэширование ответов — 262 секунды

## Запуск

Проверить строку подключения в `isp-app/appsettings.json`, выполнить миграцию `dotnet ef database update` и запустить `dotnet run`. База данных заполняется тестовыми данными автоматически при первом запуске.

## Технологии

- .NET 8
- ASP.NET Core MVC
- Entity Framework Core 8
- MS SQL Server / LocalDB
- Bootstrap 5
